          // setting up the following pins as outputs
          void setup() {
            // pin 13 is being used as an output for the first LED
            pinMode(13, OUTPUT);
            // pin 12 is being used as an output for second LED
            pinMode(12, OUTPUT);
            // pin 11 is being used as an output for the third LED
            pinMode(11, OUTPUT);
          }
          
          //loop function runs the blinking action continuously
          void loop() {
            // 5V is sent to pin 13 and turns on the LED light
            digitalWrite(13, HIGH);
            // the LED light in pin 13 stays on for 1 second
            delay(1000);
            // 0V is sent to pin 13 and turns off the LED light
            digitalWrite(13, LOW);
            // the LED light in pin 13 stays off for 1 second
            delay(1000);
            
            // 5V is sent to pin 12 and turns on the LED light
            digitalWrite(12, HIGH);
            // the LED light in pin 12 stays on for 1 second

Question 1: Draw a graph where the X axis is time and the Y axis is voltage across the LEDs. Draw a line for each of your 3 LEDs.


Question 2: How many LEDs could you blink independently with your Arduino? How much current would that draw?

A total of 20 LEDs can be blinked independently. Assuming I'm using all blue LEDs and 220 Ω resistors , 20 LEDs would draw 154 mA or 0.154 A


Calculation:

5V - 3.3V = 1.7 V
1.7 V / 220 Ω = 0.0077 A or 7.7 mA per LED
7.7 mA * 20 = 0.154 A or 154 mA total

Question 3: How fast do you need to blink your LEDs until you no longer can tell that they are blinking?

The LEDs should blink between 1 to 3 milliseconds to make it so that you can longer tell that they are blinking.
